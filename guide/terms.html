MTF 相关术语表 ​

本文收集了跨性别女性（MTF）相关的常用术语和缩写。这些术语按照类别和字母顺序排列，并提供了详细解释。

本术语表仅供参考。医疗相关内容请以专业医生的建议为准。

基础概念 ​

性别相关 ​

  • 性别认同（Gender Identity）

    • 一个人对自己性别的内在认知和感受
    • 这种认知可能与出生时被指派的性别一致，也可能不一致
    • 性别认同是一个人最真实的自我认知，应该得到尊重

  • 性别表达（Gender Expression）

    • 一个人通过外表、行为等方式展现性别的方式
    • 包括但不限于：着装、发型、举止、声音、化妆等
    • 性别表达不一定要符合社会对某个性别的刻板印象

  • 性别气质（Gender Presentation）

    • 一个人展现出的性别特质
    • 可以是阴柔的、阳刚的，或者是两者的混合
    • 与性别认同无关，不应该用来判断一个人的性别认同

  • 指派性别（Assigned Gender/Sex at Birth）

    • 出生时根据生理特征被指定的性别
    • 常见缩写：AGAB、ASAB
    • AMAB（Assigned Male at Birth）：出生时被指派为男性
    • AFAB（Assigned Female at Birth）：出生时被指派为女性

性别多样性 ​

  • 性别二元（Gender Binary）

    • 将性别简单划分为男性和女性的传统观念
    • 这种观念可能忽视了性别的多样性

  • 性别光谱（Gender Spectrum）

    • 认为性别是一个连续的光谱，而不是非此即彼的概念
    • 一个人的性别认同可能处于光谱的任何位置

跨性别相关 ​

身份认同 ​

  • MTF（Male to Female）

    • 跨性别女性，指出生时被指派为男性但性别认同为女性的人
    • 也称为 trans woman、跨性别女性
    • "跨女"是一个常用的简称，但在正式场合建议使用"跨性别女性"

  • 顺性别（Cisgender）

    • 性别认同与出生时被指派性别一致的人
    • 常见缩写：Cis
    • 例如：出生时被指派为女性且认同自己是女性的人

  • 非二元性别（Non-binary）

    • 性别认同不完全认同为男性或女性的人
    • 可能同时具有多种性别认同，或者不认同任何性别
    • 常见缩写：NB、Enby

医疗相关 ​

  • HRT（Hormone Replacement Therapy）

    • 激素替代治疗
    • MTF 主要使用：
      • 雌激素（Estrogen）：促进女性化发育
      • 抗雄激素（Anti-androgens）：抑制男性激素
    • 常见药物：
      • 雌二醇（Estradiol）
      • 色普龙（Cyproterone Acetate）
      • 螺内酯（Spironolactone）

  • SRS/GCS（Sex/Gender Confirmation Surgery）

    • 性别确认手术，也称性别重置手术
    • 主要类型：
      • 生殖器重建手术
      • 乳房手术
      • 面部女性化手术（FFS）
      • 声带手术
    • 不是所有跨性别者都需要或想要进行手术

诊断相关 ​

  • 性别焦虑（Gender Dysphoria）

    • 因性别认同与指派性别不一致而产生的不适感
    • 可能表现为：
      • 对自己身体特征的强烈不适
      • 社交场合中的焦虑
      • 被误认性别时的痛苦

  • 性别欣悦（Gender Euphoria）

    • 当性别表达与性别认同一致时产生的愉悦感
    • 例如：
      • 被正确使用代词时的开心
      • 穿着心仪的衣服时的满足感
      • 激素治疗带来的身体变化带来的喜悦

社群用语 ​

日常用语 ​

  • 姐妹

    • MTF 社群中常用的互相称呼
    • 表示认同和支持
    • 在正式场合可能不适用

  • 代词（Pronouns）

    • 用来指代某人的人称代词
    • 常见选择：
      • 她/她的（she/her）
      • 他/他的（he/him）
      • 它/它的（they/them，用于英语中性指代）
    • 应尊重每个人选择的代词

  • 出柜（Coming Out）

    • 向他人公开自己的性别认同
    • 可以选择：
      • 完全出柜：向所有人公开
      • 部分出柜：只向特定人群公开
    • 出柜是个人选择，不应该被强迫

过渡相关 ​

  • 过渡期（Transition）

    • 从原有性别角色转变到目标性别角色的过程
    • 包括：
      • 社会转变：改变称谓、着装等
      • 医疗转变：激素治疗、手术等
      • 法律转变：修改身份证件等
    • 每个人的过渡过程都是独特的

  • Passing

    • 能够被他人自然地认作目标性别
    • 不是所有跨性别者的必要目标
    • 与 passing 相关的压力可能带来负面影响

  • 全职生活体验（Real Life Experience, RLE）

    • 以目标性别身份生活的经历
    • 在某些地区可能是获得医疗资源的前提条件
    • 对其必要性存在争议

社交媒体标识 ​

  • 跨性别符号

    • ⚧：跨性别符号，最常用的身份标识之一
    • 🏳️‍⚧️：跨性别旗帜（蓝白粉）
    • 🍥：在中文社群中表示正在进行 HRT 治疗
    • 🦋：蝴蝶符号，有时用来表示跨性别身份
    • 💊：药丸符号，常用来表示正在进行激素治疗

  • 用户名特征

    • 常见组合：名字 + ⚧ + 🏳️‍⚧️
    • 示例：Alice⚧🏳️‍⚧️、小明🏳️‍⚧️🍥
    • 一些用户会在简介中标注代词（如：she/her）

  • 常用标签

    • #mtf：最基础的身份标签
    • #transgender：国际通用标签
    • #跨性别：中文社群标签
    • #HRT：激素治疗相关讨论
    • #跨性别女性：正式场合使用的标签

社群交流提示 ​

    • 查看相关标签下的讨论

  • 安全建议

    • 谨慎分享个人信息
    • 可以使用小号或匿名账号
    • 警惕可疑的私信和骚扰

  • 社交礼仪

    • 尊重他人的隐私
    • 不要随意询问他人的过渡细节
    • 避免使用可能引起不适的称呼
    • 遇到恶意评论时及时举报和屏蔽

注意

社交媒体上的身份标识仅供参考，不是所有跨性别者都会使用这些符号。同时，使用这些符号的人也不一定都是跨性别者。请尊重每个人的隐私和选择。

权益相关 ​

法律权益 ​

  • 性别标记（Gender Marker）

    • 身份证件上的性别信息
    • 修改程序因地区而异
    • 可能需要医疗证明或手术证明

  • 反歧视保护

    • 保护跨性别者免受就业、教育等方面歧视的法律
    • 不同地区的保护程度不同

医疗权益 ​

  • 知情同意（Informed Consent）
    • 在充分了解风险和效果的基础上做出医疗决定
    • 一些诊所采用知情同意模式提供 HRT

注意事项 ​

  1. 这些术语仅供参考，每个人对术语的理解和使用可能有所不同
  2. 与他人交流时应尊重对方的称谓偏好
  3. 部分术语可能带有医疗或病理化的历史，使用时需要注意场合
  4. 术语会随时间演变，本表将定期更新
  5. 在不同语境下，某些术语可能有不同的含义
  6. 避免使用可能冒犯他人的过时术语
